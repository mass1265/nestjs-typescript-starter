Quick memory tip: "Relative reserves space & nudges; absolute removes space & places exactly." n
  • Senior:"Set position: relative on a container to create a positioning context for absolutely positioned children; otherwise absolute elements are positioned relative to the viewport."
  • Mid:"An absolutely positioned element is removed from document flow and positioned relative to its nearest positioned ancestor; relative keeps the element in-flow and applies visual offsets."
  • Junior:"Use relative to nudge an element while keeping its space; use absolute to place it exactly where you want without affecting other elements."
